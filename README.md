# GiftItem[抽奖插件]

## 用途
- 将手中物品进行抽奖

## 版本
- Mc：1.21.x
- 构建所需java版本: java21[jdk-21]

## 说明
- /gift all [将手中物品给予所有人]
- /gift one [将手中物品抽一个人送出]
